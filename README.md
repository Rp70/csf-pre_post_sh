# ConfigServer Security & Firewall (CSF) - csfpre.sh / csfpost.sh

## Install
Installation is quite straightforward:

```
cd /usr/local/src
sudo git clone https://github.com/Rp70/csf-pre_post_sh.git
cd csf-pre_post_sh
sudo sh install.sh
```
